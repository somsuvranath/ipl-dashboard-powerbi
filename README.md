# 📊 Sports Analytics Dashboard (Power BI)


## 🎯 Live Interactive Dashboard

> **🔗 Click here to view the live dashboard:** https://somsuvranath.github.io/ipl-dashboard-powerbi/

---

## 📌 Problem Statement

Large sports datasets contain valuable insights, but without proper modelling and visualization, extracting meaningful patterns is difficult.

---

## 🎯 Objective

To build an interactive dashboard that analyzes IPL data and demonstrates advanced data analytics and visualization capabilities.

---

## 🛠️ Approach

### Data Processing

* Cleaned and transformed historical IPL data (2008–2024)
* Structured datasets for analysis using Power Query

### Dashboard Development

* Built interactive visuals using Power BI
* Created DAX measures for:

  * Player performance
  * Match outcomes
  * Toss decision analysis

---

## 📊 Key Insights of 2024

* Teams choosing to field first had a **higher probability of winning (~77%)**
* Certain players consistently outperform across seasons
* Match outcomes show strategic dependence on toss decisions

---

## 💼 Relevance to ESG/Data Roles

This project demonstrates:

* Data modelling and transformation
* KPI development using DAX
* Dashboard storytelling

These capabilities are directly applicable to ESG analytics and reporting.

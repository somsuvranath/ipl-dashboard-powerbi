# IPL-dashboard-powerbi
IPL Analysis Dashboard built using Power BI
## 📌 Project Overview
An interactive Power BI dashboard analyzing IPL (2008 to 2024)seasons data
including player stats, team performance, and toss-decision impact.

## 🔍 Key Insights( based on 2024, years can be changed in slicer) 
- 🏆 Tournament Winner: Kolkata Knight Riders
- 🧡 Orange Cap: V Kohli — 741 Runs
- 💜 Purple Cap: JJ Bumrah — 30 Wickets
- 📊 Teams fielding first won **77% of matches**
- 💥 Total Runs: 25,971 | Total Wickets: 883

## 📊 Features
- KPI Cards: Winner, Runs, Wickets, 4s, 6s, Player of Match
- Player Panels: Batsman & Bowler stats with dropdown filters
- Top 5 Batsmen & Bowlers bar charts
- Team win breakdown (runs & wickets)
- Toss decision win % donut chart
- Year filter slicer (2024)

## 🛠️ Tools Used
| Tool | Purpose |
|------|---------|
| Power BI Desktop | Dashboard building |
| DAX | Custom measures & KPIs |
| Power Query | Data cleaning & transformation |
